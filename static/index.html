• /buckets
    • Return a list of publicily readable buckets
  • /details/<bucket>
    • Return a dict with details of the bucket:
      • bucket: full bucket path
      • bytes-used: bucket size in bytes
      • name: name of the bucket
      • object-count: number of objects in the bucket
      • site: OSN site
  • /object-list/<bucket>
    • Return a list of all objects in the bucket with details:
      • etag: MD5 hash of the object
      • key: name and path of the object
      • last-modified: last modified date and time
      • size: size of the object in bytes
      • url: public URL of the object
